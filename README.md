# Flow — RSS Reader for Android

A clean, immersive RSS reader built with Jetpack Compose.

## Features
- Full-text article fetching with Readability4J
- Immersive reader mode (edge-to-edge, auto-hide bars)
- Georgia typography (Pocket-style)
- Gemini Nano AI summarization
- Text-to-speech
- Bookmark & share

## Tech Stack
Jetpack Compose · Room · Hilt · Retrofit · Readability4J · Jsoup · Coil

## Requirements
Android 16+ (Pixel 8 Pro target)
